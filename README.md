
1. Ubicación de Archivos

Crear carpeta vehiculos en C:\xampp\htdocs\vehiculos

Añadir dentro las carpetas: api, config, includes, public (con sus respectivos archivos .php)

2. Configurar el Entorno Local (XAMPP)

Abrir panel de XAMPP.

Iniciar los servicios de Apache (80, 443) y MySQL (3306).

3. Crear e Implementar la Base de Datos

En el navegador, ir a: http://localhost/phpmyadmin

Ir a la pestaña superior SQL.

Pegar todo el código del archivo database.sql.

Clic en Continuar (esto crea la base de datos control_transito y las tablas automáticamente).

4. Abrir el Sistema

En el navegador, ir a: http://localhost/vehiculos/public/index.php
